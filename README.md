# Hotel Review Prediction using BERT + MLP

This project focuses on hotel review classification and score prediction using deep learning and Natural Language Processing techniques.

## Features
- BERT embeddings
- Multi-task learning
- Review classification
- Score prediction
- Neural network models

## Technologies Used
- Python
- NLP
- Deep Learning
- BERT
- TensorFlow / PyTorch
- Jupyter Notebook

## Project Overview
The project combines text-based embeddings and neural network architectures to analyze hotel reviews and predict both review categories and review scores.
